# Hacker News APP

A tech news website, developed for the JavaScript Advanced project of [Start2Impact](https://www.start2impact.it). <br />
This app uses [Hacker News API](https://github.com/HackerNews/API) to get the news.

# Used technologies:

* HTML, CSS, JS
* Axios
* Node.js
* Webpack
